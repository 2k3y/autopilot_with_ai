# 🚁 AgroTech AI: Drone Autopilot

Система автономного мониторинга сельскохозяйственных полей. Дрон летит по заданному маршруту («змейка») и в реальном времени анализирует поверхность с помощью нейросети MobileNetV2, выявляя участки без посевов.

## 🛠 Особенности проекта
* **AI Analysis:** Использование CNN для классификации поверхности (Planted / Empty).
* **Smart Navigation:** Автоматическое построение маршрута и точная посадка по П-регулятору.
* **Easy Config:** Все параметры полета и связи вынесены в отдельный файл `config.py`.

## 📂 Структура
* `autopilot_with_ai.py` — основной скрипт миссии.
* `config.py` — настройки (IP, высота, координаты).
* `requirements.txt` — список необходимых библиотек.
